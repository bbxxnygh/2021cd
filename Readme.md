## 中共「柏林牆時刻」即將來臨

* 誰主白宮與中美關係變化
⸻ 中美緊張關係出現緩解還是加劇

⸻中共若武力攻台 美日是否會參戰

* 黑天鵝會否衝擊世界秩序
⸻世界秩序是否會有顛覆性新變化
* 在線互動解答網友提問
